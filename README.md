# internet_court
日常の小さな「罪」を匿名で投稿し、他の参加者全員でリアルタイムに裁くエンタメ型Webアプリケーションです。 各投稿に対して「有罪」「無罪」の投票や、共感を示す「心中お察し」ボタンによるインタラクションが可能。最大の特徴は、単なる多数決ではなく、投票割合を反映したルーレット演出によって最終判決が下るスリリングなシステムです。 バックエンドにNode.jsとExpress、リアルタイム通信にSocket.io、データベースにSQLiteを採用。投票数や共感数に基づくリアルタイムランキング機能も備えています。インターネットならではの「世論のうねり」を、直感的かつエンターテインメント性の高いUI/UXで疑似体験できる作品です。
